# Проект мониторинга депозитных ставок банков

## Описание
Система для сбора, обработки и визуализации данных о депозитных ставках топ-10 банков России.

## Технологический стек
- Apache NiFi - сбор данных
- Apache Airflow - оркестрация ETL
- PostgreSQL - хранилище данных
- dbt - трансформация данных
- Redash - визуализация
